static public void callIteratively (number totalDuration, number timesToCall, Function stepFunction, Function endFunction, Object context)

Attempts to invoke a function iteratively in the background a specific number of times within a specific duration. This might not complete in the specified duration. The functions passed in should be short functions that return as quickly as possible. If you are using long functions, use the recursive setTimeout trick by-hand instance.

Parameters

number totalDuration

The duration in milliseconds.
number timesToCall

The number of times to call the function.
Function stepFunction

The step function to call each iteration.
Function endFunction

The function to invoke at the end. (Optional)
Object context

The context for the function calls (sets the 'this' parameter). (Optional)

Return Value

void
