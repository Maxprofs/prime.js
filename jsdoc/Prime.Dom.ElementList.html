each(iterationFunction, context) → {Prime.Dom.ElementList}

Iterates over each of the Prime.Dom.Element objects in this ElementList and calls the given function for each one. The 'this' variable inside the function will be the current Prime.Dom.Element unless a context value is provided when calling this function. The function can optionally take two parameters. The first parameter is the current element. The second parameter is the current index.
