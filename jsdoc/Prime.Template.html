generate(parameters) → {string}

Generates a String from the given parameterHash. Provide a hash of String keys to values. Keys can be regular text strings, in which case it will look for and replace #{key} as with the value. You can also make the key a String "/key/", which will be converted to a Regex and run. For the value you can provide a straight up String, int, etc, or you can provide a function which will be called to provide the value
Parameters:

parameters Object An object that contains the parameters for the template to replace.
