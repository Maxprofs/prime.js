public Prime.Document.ElementList each (Function iterationFunction, Object context)

Iterates over each of the Prime.Document.Element objects in this ElementList and calls the given function for each one. The 'this' variable inside the function will be the current Prime.Document.Element unless a context value is provided when calling this function. The function can optionally take two parameters. The first parameter is the current element. The second parameter is the current index.

Parameters

Function iterationFunction

The function to call.
Object context = the-current-element

The context for the function call (sets the this variable). (Optional)
