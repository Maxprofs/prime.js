Document.Element.js at line 987
public Prime.Document.Element show (string displayValue)

Shows the Element by setting the display style first to empty string. After this, the elements computed style is checked to see if the element is still not visible. If that is true, the element must have a CSS style defined in a stylesheet that is setting it to display: none. In this case, we determine if the element is a block level element and either set the display to 'block' or 'inline'.

Parameters

string displayValue

The display value to use for the show. This defaults to the W3C standard display setting depending on the type of element you are showing. For example, INPUT is inline and DIV is block. (Optional)

Return Value
